# User Registration & Login System using Django

## Project Overview

This project is a User Registration and Login System developed as a Full-Stack web application.

The main purpose of this project is to understand how frontend, backend, and database components connect and work together.

## Technologies Used

* Python
* Django
* MySQL
* HTML
* CSS
* JavaScript

## Features

* User Registration
* User Login Authentication
* Database Integration
* Backend Logic using Django
* User Data Management
* Complete Workflow Testing

## Project Workflow

Frontend (HTML, CSS, JavaScript)
↓
Django Backend (Python)
↓
MySQL Database
